# Blog Management App

## Install

pip install django
pip install pillow

python manage.py migrate
python manage.py createsuperuser
python manage.py runserver

## Cara Menjalankan Project

git clone LINK_REPO

cd django-blog-app

python -m venv venv
venv\Scripts\activate  # windows

pip install -r requirements.txt

python manage.py migrate
python manage.py createsuperuser
python manage.py runserver

Fitur
- Landing Page
- Artikel Publik
- Search Artikel
- Pagination
- Admin Login
- Dashboard Admin
- CRUD Artikel

Teknologi
- Django
